## Принципи

1. **Nan-Data First**: Кожен байт несе сенс.
2. **Database-as-Truth**: Конфігурація живе в базі даних, не в коді.
3. **One Logic — Many UI**: Єдина доменна модель для всіх середовищ.
4. **Document-as-Truth**: Будь-який контент — це документ у базі.
