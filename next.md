# 📝 Наступні задачі (Next)

## Контекст (Що вже зроблено в останній сесії)

- **OLMUI Архітектура**: Усі ключові інтерактивні CLI-застосунки (`InspectorApp`, `AuditorModel`, `GraphCommand`, `I18nCliApp` / `App`) успішно переведено на використання єдиного `ModelAsApp`. Додано типізації через `ModelAsAppOptions`.
- **Ізоляція та Завантаження (`bootstrapApp`)**: Виконано масивний рефакторинг універсального раннера додатків. Логіку розділено на дрібніші утиліти: `readParams`, `resolveDsn`, `readMounts`, `resolveModel`. Покращено роботу з TTY (`process.exit` тепер має таймаути для коректного виводу).
- **OLMUI Адаптери (`InputAdapter`)**: Реалізовано асинхронне "ліниве" завантаження всіх промптів (через `await import()`), що дозволяє сильно розвантажити пам'ять. Тепер всі виклики промптів проходять через централізований `executePrompt`, який вміє обробляти `UI_SNAPSHOT` та автоматизовані інжекти відповідей.
- **Типізація Контенту (`Content.js`)**: Додано Zero-Runtime-Cost Typings (`h1`, `p`, `table`, `input`, `form` тощо) до `Content` для ідеального автокомпліту в `.js`.
- **План `Nan0SpecRunner`**: Оновлено (і деталізовано) `seed.md` у пакеті `@nan0web/test` — підготовка до TDD за допомогою строгих стрімів інтенцій (`Nan0SpecAdapter`).
