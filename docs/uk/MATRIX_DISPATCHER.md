## 🔐 2. Протокол Суверенної Ідентичності (Auth)

Користувач не "реєструється" в додатках. Додатки "просять доступу" до Користувача.

### Потік Авторизації (Auth Flow)

1.  **Запит**: Додаток (напр., Адвокати) просить: `yield Prompt('Auth', { scope: ['email', 'payment'] })`.
2.  **Посередництво**: **Auth** сервіс (на телефоні/сервері користувача) перехоплює цей інтент.
3.  **Згода**: Користувач бачить: "Адвокати хочуть знати ваш Email та списати 100 грн".
4.  **Мандат**: Користувач підтверджує (FaceID/Passkey) -> Auth видає тимчасовий Токен.

## 🌐 3. Соціальна Федерація (Web3)

Кожен додаток є також **видавцем контенту**.

- **Personal Portal**: Публікує статті та думки.
- **Advocates**: Публікує правові прецеденти.
- **Dynamo**: Публікує розклад подій.

**Агрегатор Стрічки (The Feed Aggregator)**:
Матриця збирає всі ці потоки в єдину **"Картину Світу"** для користувача.
`yield* flow(Aggregator.run)` підтягує дані з усіх авторизованих джерел без централізованих алгоритмів.

## ⚖️ 4. Спецпроект: Національний Аудит (Індекс Корупції)

Інструмент для **Прозорого Правосуддя**.

- **Індексація**: AI сканує Prozorro/OpenData та фіксує аномалії (напр., "Ціна > 300% від ринкової").
- **Підготовка**: Система генерує **чернетку скарги** для юристів. Вона НЕ судить, вона готує доказову базу.
- **Людський Контроль**: Кваліфікований Адвокат перевіряє чернетку та підписує її своїм цифровим підписом. **AI — це перо, Людина — це рука.**
