# NaN0Web – децентрализованная крипто‑анархическая платформа

## Обзор

**NaN0Web** — открытый проект, объединяющий философию свободных людей, логику программирования и украинский язык.  
Сеть гарантирует **владение данными** их создателей и взаимодействие с языковыми моделями на украинском, что повышает прозрачность и безопасность.

## Исторический контекст

- В 2014 году автор переехал из Запорожья во Львов, чтобы изучать украинский язык и культуру, осознав, что язык формирует мышление и что украинский — логичный и не испорченный имперскими традициями.  
- После агрессии/войны стало ясно, что борьба против империализма начинается с языка: язык формирует мышление и быт, а на русском языке ничего хорошего не получалось.  
- Информация — высшая ценность, создаваемая человеком.  
- Важность владения информацией: **данные и знания должны оставаться у владельца**, а не в социальных сетях, которые претендуют на право удалять или блокировать эти данные.

## Философия

1. **Свобода языка** — украинский язык не навязывает имперские названия (например, названия месяцев) и позволяет точно формулировать мысли.  
2. **Свобода кода** — код открыт, доступен каждому сознательному человеку, без ограничений.  
3. **Владение данными** — информация принадлежит её создателю; платформа не собирает и не монетизирует данные сторонних сервисов.  
4. **Децентрализация‑крипто‑анархия** — сеть работает без централизованных серверов, используя криптографические протоколы для защиты и аутентификации.  
5. **Прозрачность = безопасность** — открытый код позволяет любому проводить аудит, исправлять уязвимости и обеспечивать квантовую устойчивость.

## Почему именно украинский язык?

Украинский обладает чёткой грамматикой и структурой, позволяющей формулировать логические утверждения без лишних метафор. В отличие от русского, он не искажает логику и не привязан к имперским названиям. Хороший пример — названия месяцев, которые в украинском основаны на природных явлениях, а не на именах императоров или богов.

## Архитектура платформы

- **Языковые модели** — взаимодействуют на украинском уровне инструкций, обеспечивая точность и понятность.  
- **Пакеты NaN0Web** — модульная система, включающая Core, Browser/Mobile, React‑компоненты и др.  
- **Тестирование** — 100 % покрытие unit‑тестами (`node:test`), `vitest` для React и **Playground** для автоматизированных демонстраций.  
- **Типизация** — JavaScript‑typedef компилируется в чистый TypeScript `d.ts`, предоставляя автодополнение в IDE.  
- **Документация** — генерируется из тестов, доступна на украинском и английском.

## Децентрализованная крипто‑анархическая сеть

- **Поддержка блокчейна** — данные шифруются и хранятся в распределённых узлах, предотвращая цензуру.  
- **Квантовая устойчивость** — внедрены современные пост‑квантовые алгоритмы, подготовленные к будущим угрозам.  
- **Прозрачный протокол** — открытые API позволяют кому‑угодно подключать свои сервисы без центрального контроля.

## Владение данными

1. **Авторское право** — данные остаются у их создателя.  
2. **Контроль доступа** — пользователь сам решает, кто может читать или изменять данные через децентрализованные ACL.  
3. **Открытость** — данные могут быть подключены к другим сетям через открытые протоколы без потери собственности.

## Безопасность и прозрачность

- **Открытый код** — любой может провести аудит, исправить уязвимости и повысить безопасность.  
- **Публичность** — публикация кода — лучшая защита: без скрытых намерений, без «чёрных ящиков».  
- **Квантовая устойчивость** — подготовка к будущим вычислительным угрозам через пост‑квантовые криптографические схемы.

## Как присоединиться

1. **Клонировать репозиторий**  
   ```bash
   git clone https://github.com/nan0web/monorepo.git
   cd monorepo
   pnpm install   # или npm / yarn
   ```
2. **Запуск Playground**  
   ```bash
   pnpm play   # автоматизированные примеры
   ```
3. **Запуск тестов**  
   ```bash
   pnpm test:all
   ```
4. **Создавать новые пакеты** — следуя шаблону `nan0package` (`src/`, `test/`, `play/`, `README.md`).
