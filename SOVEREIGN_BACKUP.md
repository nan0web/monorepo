# Sovereign Backup Strategy (Суверенна стратегія бекапу)

## 🛡 Філософія

Дані — це цифрова плоть нашого творіння. Ми не покладаємося на хмари, що нам не належать. Ми володіємо копією кожного байта.

## 📁 Структура сховища

- **Код:** GitHub (origin) + Зовнішній диск (daily mirror).
- **Статика (1.5 ГБ+):** Dedicated Assets Server + Локальне дзеркало.
- **БД:** Автоматичні дампи YAML/JSON кожні 6 годин.

## 🚀 Протокол відновлення (DRP)

1. **Клонування ядра:** `git clone git@github.com:industrialbank/bank.git`
2. **Запуск майстра:** `node init.js --mode=dev`
3. **Синхронізація статики:** `pnpm run assets:sync`

## 📅 Графік бекапів

- **Daily:** `rsync` на зовнішній диск.
- **Weekly:** Архівування на віддалений сервер (Sovereign Cloud).
