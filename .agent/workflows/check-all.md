// turbo-all

1. **Генерація черги**: Запусти `node bin/workspace-status.js --requests` і прочитай вивід.

2. **Ітерація по пакетах**: Для кожного пакета з pending tasks (з виводу workspace-status):
   - Перейди в директорію пакета
   - Прочитай `next.md` або `REQUESTS.md`
   - Запусти `pnpm test:all` (якщо є такий скрипт)
   - Якщо тести падають — виправ використовуючи TDD (Red→Green protocol)
   - Якщо тести проходять — повідом і переходь до наступного пакета

3. **Звіт**: Після обходу всіх пакетів — створи артефакт `workspace-health-report.md` з:
   - Таблицею пакетів та їх статусом (pass/fail/skip)
   - Переліком виправлених проблем
   - Переліком невирішених блокерів

4. **Не комітити автоматично**: Лише показати що змінилось. Коміт по `/commit` окремо.
